# Windows 系统通知设置指南

## 功能说明

这个应用现在包含以下通知功能：

✅ **提前15分钟提醒**：当任务即将在15分钟内开始时，你会收到一条桌面通知，显示剩余的时间。

✅ **任务开始提醒**：当任务时间到达时，你会收到一条立即提醒的桌面通知。

✅ **音频提醒**：每次通知都会播放一个简短的音调来吸引你的注意。

✅ **精准检查**：应用每10秒检查一次任务时间，确保不会错过任何提醒。

---

## 如何启用通知

### 方法 1：首次访问（推荐）

1. 刷新页面或首次打开应用
2. 浏览器会弹出权限请求对话框
3. 点击 **"允许"** 按钮
4. 你会看到应用顶部的状态从 "⚠️ 通知未启用" 变为 "✅ 通知已启用"

### 方法 2：从浏览器设置启用（如已拒绝）

#### Microsoft Edge 使用者：
1. 打开应用所在的 Edge 浏览器
2. 点击 URL 栏左侧的"通知"禁用图标（看起来像个铃铛，可能被叉掉）
3. 选择 **"允许"** 或 **"始终允许"**
4. 刷新页面

#### Google Chrome 使用者：
1. 打开应用所在的 Chrome 浏览器
2. 点击 URL 栏右侧的"锁"图标
3. 找到"通知"选项
4. 将其改为 **"允许"**
5. 刷新页面

#### 其他 Chromium 浏览器：
类似 Chrome 的步骤操作

## Windows 系统设置

如果你想调整 Windows 的通知显示方式：

1. **打开 Windows 设置**：按 `Win + I`
2. **进入通知设置**：系统 → 通知和操作
3. **调整设置**：
   - 启用"获取通知"
   - 启用"在锁屏上显示通知"（可选）
   - 在"通知"部分配置浏览器(Edge/Chrome)的通知权限

1. **第一个通知**（15分钟内）：

## 常见问题

**Q: 为什么看不到通知？**
- 检查浏览器是否已授予权限（见上方设置步骤）
- 检查 Windows 通知中心是否被关闭
- 检查浏览器是否在后台运行（应用需要保持活跃以继续检查）

**Q: 可以自定义提醒时间吗？**
- 当前固定为提前 15 分钟和任务开始时刻
- 后续版本可能支持自定义提醒间隔

**Q: 浏览器关闭了，还能收到通知吗？**
- 不能。如果你想要无缝的通知，可以考虑 Electron 版本或使用浏览器的后台运行功能

**Q: 通知有声音吗？**
- 是的，每个通知都会播放一个简短的音调
- 音量取决于你的系统音量和浏览器的音量设置
- 某些情况下可能不会播放（例如浏览器被静音）

## 调试

如果仍然无法看到通知，请检查浏览器控制台（F12 → Console）：

- `✅ 已获得通知权限` - 权限已正确授予
- `⚠️ 用户已拒绝通知权限` - 需要在浏览器设置中重新启用
- `浏览器不支持通知功能` - 浏览器不兼容
