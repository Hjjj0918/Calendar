# Changelog

本文档记录 Reminder-Plugin 的发布历史，遵循「版本号 + 日期 + 变更分类」格式。

## [v0.2.0] - 2026-03-24

### Added
- 新增「任务列表」「新建任务」「已归档任务」三个按钮作为统一入口。
- 新增弹窗式任务管理：点击入口按钮后弹出操作小框，保留原有任务信息展示与填写能力。
- 支持在任务列表中点击“编辑”后直接打开任务表单弹窗并回填原数据。

### Changed
- 页面由常驻分栏改为按需打开，首屏信息更聚焦。
- 优化长列表滚动体验：弹窗内任务列表区域增加高度限制与滚动条，任务数量较多时不再拉长整页。

### Fixed
- 保持通知行为稳定：提前 15 分钟提醒、任务开始提醒与音频提醒逻辑在本次重构后不变。

## [v0.1.0] - 2026-03-19

### Added
- 在每个任务底部操作区增加“归档”按钮。
- 在“已归档任务”中增加“取消归档”按钮。

### Changed
- 任务列表组件支持自定义标题、空状态文案、状态标签。
- 归档与取消归档复用统一按钮渲染逻辑，减少重复代码。
- 看板统计聚焦“未归档任务”，更符合日常执行视角。
