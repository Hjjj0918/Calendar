# Reminder-Plugin

一个基于 React + Vite 的时间段任务提醒插件，用于管理任务开始/结束时间，并在关键节点发送提醒。

## 快速开始

1. 进入项目目录：`cd reminder-plugin`
2. 安装依赖：`npm install`
3. 启动开发环境：`npm run dev`

如果在 Windows PowerShell 遇到脚本执行策略限制，可使用：

- 安装依赖：`npm.cmd install`
- 启动开发：`npm.cmd run dev`

## 当前已实现功能

- 任务管理：支持创建、编辑、删除任务。
- 时间段任务：每个任务包含开始时间和结束时间。
- 任务状态识别：自动展示“未开始 / 进行中 / 已结束”。
- 通知提醒：
	- 任务开始前 15 分钟发送预提醒。
	- 到达开始时间发送即时提醒。
	- 浏览器通知支持权限检测与引导。
- 本地持久化：任务数据保存在 LocalStorage，刷新页面后不丢失。
- 归档能力：
	- 任务列表中可一键归档任务。
	- 已归档任务单独分栏展示。
	- 已归档任务支持“取消归档”回到主列表。
	- 已归档任务不参与提醒触发。
- 数据看板：展示“总任务 / 进行中 / 待开始”统计。
- 交互反馈：关键操作显示 Toast 提示。
- 响应式布局：兼容桌面和移动端浏览。

## 后续可扩展方向

- 增加“全部归档 / 全部取消归档”批量操作。
- 支持任务搜索、状态筛选、时间排序切换。
- 增加重复任务（每日/每周）与提醒提前时间自定义。
- 引入导出与备份能力（JSON/云同步）。
